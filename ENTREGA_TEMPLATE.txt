RESUMO DO PROJETO
--------------------------------------------------------------------------------

OBJETIVO:
Desenvolver um modelo de Machine Learning para auxiliar médicos e médicas
a prever se uma pessoa pode ter obesidade, utilizando dados de questionários
e características antropométricas.

METODOLOGIA:
- Feature Engineering: Criação de IMC, Risk Score e outras features derivadas
- Múltiplos algoritmos testados: Gradient Boosting, Random Forest, SVM, KNN, etc.
- Validação cruzada (5-fold) para seleção do melhor modelo
- Pipeline completo de pré-processamento para produção
